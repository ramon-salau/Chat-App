# Chat-App
A real-time chat application using ASP.Net Web API, WPF &amp; SignalR. Available for Web and Desktop

## Features
* Group chat
* Private chat
* Photo message
* Basic Emojis
* Chat Rooms

## Technologies used
1. WPF(Windows Presentation Foundation) in C#
2. ASP.Net Core Web API
3. SignalR
4. ASP.Net Framework
5. SQL Server Database
6. Bootstrap
7. Javascript
